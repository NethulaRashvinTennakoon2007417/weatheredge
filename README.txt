# ⚡ Weather Edge Analyzer

Real-time bracket probability analyzer for Polymarket weather markets.
Finds mispriced brackets before the market corrects.

---

## Quick Start

### Windows
Double-click `START_WINDOWS.bat`

### Mac
1. Right-click `START_MAC.sh` → Open With → Terminal
   (First time only: in Terminal run `chmod +x START_MAC.sh`)
2. Or open Terminal, drag the file in, press Enter

### Manual (any OS)
```
pip install -r requirements.txt
python launch.py
```

## Troubleshooting

**"Python not found"**
Download Python from python.org — check "Add Python to PATH" during install

**"Access denied" with a valid code**
Make sure you copy the full code including the dot and characters after it

**App opens but browser doesn't load**
Manually go to http://localhost:5000 in your browser

**Any other issue**
DM @NethulaRashvin on X with a screenshot
